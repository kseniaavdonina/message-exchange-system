## Описание системы

РИТМ — это корпоративная система обмена сообщениями с поддержкой файловых вложений неограниченного размера.

### Основные возможности

- Отправка и получение сообщений с вложениями
- Поддержка файлов любого размера (через MinIO S3)
- Управление пользователями и ролями (USER / ADMIN)
- Горизонтальное масштабирование (автоматическое создание контейнеров)
- REST API для интеграции с внешними системами (1С, CRM, ERP)
- Хранение сессий в Redis (централизованное)
- Асинхронная доставка уведомлений через ActiveMQ

### Технологический стек

| Компонент | Технология |
|-----------|------------|
| Язык | Java 17 |
| Фреймворк | Spring Boot 2.7.14 |
| База данных | PostgreSQL 16 |
| Файловое хранилище | MinIO (S3-совместимое) |
| Очереди сообщений | ActiveMQ 5.15.9 |
| Кэш и сессии | Redis |
| Балансировка | Nginx |
| Контейнеризация | Docker / Docker Compose |

## Требования

### Минимальные системные требования

- **CPU**: 2 ядра
- **RAM**: 4 ГБ (рекомендуется 8 ГБ)
- **Диск**: 10 ГБ свободного места
- **ОС**: Linux / Windows / macOS

### Программное обеспечение

- Docker (версия 20.10+)
- Docker Compose (версия 2.0+)
- Java JDK 17
- Maven 3.9+
- Git (опционально)

```bash

cd ritm-system

### Шаг 2: Сборка приложения

```bash
mvn clean
mvn package

### Шаг 3: Запуск всех сервисов

### Шаг 4: Проверка статуса контейнеров

```bash

### Шаг 5: Просмотр логов (при необходимости)
